
# Prayer Times Display App

This Python application uses tkinter to display daily prayer times based on selected cities and calculation methods using the AdhanPy library.

## Features

- Shows prayer times for today: Fajr, Sunrise, Dhuhr, Asr, Maghrib, Isha.
- Choose a city and calculation method from dropdown menus.

## Setup

1. Clone the repository or download `main.py`.
2. Install dependencies:

   ```bash
   pip install adhanpy
   ```

3. Set environment variables or configure sensitive data (e.g., API keys).

## Usage

1. Run `main.py`:

   ```bash
   python main.py
   ```

2. Select a city and calculation method.
3. Click **Update Prayer Times** to display times.
4. Click **Change Calculation Method** to switch methods.
